About the project

Karo Harvest was a student project that lasted half a semester. Its goal was to create a concept of a future in 30 years, and create a game that could be played by people in that time using VR or AR technology. Working with VR for the first time was challenging, but not only did I manage to program various interactions, the product received maximum grade from the teachers.

My Contributions

VR Hands

Imported Oculus hands from Meta, then animated and scripted them.

Dial Interaction

Implemented dial that controls the UI, and can react to how user rotates it.

Humidity Changer

Communicates with the dial to change humidity for plants to grow.
